# Haeni's Interactive Portfolio

A sophisticated, motion-rich React portfolio website showcasing frontend development skills through immersive user experiences. Built with sun/flower blooming themes and premium visual polish.

## Tech Stack

- **Frontend**: React 18+ with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom design system
- **Animation**: Framer Motion (primary) + GSAP (selective)
- **Smooth Scrolling**: Lenis
- **Icons**: Lucide React

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Lint code
npm run lint

# Format code
npm run format

# Type check
npm run type-check
```

## Project Structure

```
src/
├── components/          # Reusable UI components
│   ├── ui/             # Base design system components
│   ├── layout/         # Layout-specific components
│   └── interactive/    # Animation-heavy components
├── sections/           # Page section components
├── hooks/              # Custom React hooks
├── utils/              # Utility functions and helpers
├── data/               # Content data and mock content
├── types/              # TypeScript type definitions
└── styles/             # Global styles and theme
```

## Features

- 🌻 Sun/flower themed design with warm, blooming aesthetics
- ✨ Rich motion system with scroll-triggered animations
- 🎯 Featured projects spinner with 3D-inspired interactions
- 📱 Responsive design with mobile-first approach
- ♿ Comprehensive accessibility with reduced motion support
- ⚡ Performance-optimized animations (60fps target)
- 🎨 Premium visual polish suitable for portfolio review

## Design Principles

- **Performance-First**: Transform/opacity animations for smooth 60fps
- **Accessibility**: Built-in reduced motion and keyboard navigation
- **Maintainable**: Clean component architecture without over-engineering
- **Responsive**: Seamless experience across all devices
- **Content-Flexible**: Easy content updates through structured data models
